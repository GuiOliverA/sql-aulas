# 🗄️ Masterizando SQL - Do Zero ao Relacional

Este repositório contém todos os scripts, exercícios e anotações que desenvolvi durante o curso de **MySQL** do **Curso em Vídeo (Prof. Gustavo Guanabara)**. Aqui documento minha evolução na manipulação de bancos de dados relacionais, preparando a base para meus estudos avançados em **Big Data**.

---

## 🚀 O que este repositório cobre?

O projeto está organizado por módulos de aprendizado, focando em:

- **DDL (Data Definition Language):** Criação e alteração de estruturas de tabelas.
- **DML (Data Manipulation Language):** Inserção, atualização e exclusão de registros.
- **DQL (Data Query Language):** Consultas complexas, filtros inteligentes e funções de agregação.
- **Modelagem Relacional:** Implementação de Chaves Primárias (PK), Chaves Estrangeiras (FK) e relacionamentos 1:1, 1:N e N:M.
- **Junções Avançadas:** Uso de `INNER JOIN`, `LEFT JOIN` e `RIGHT JOIN` para integração de dados.
